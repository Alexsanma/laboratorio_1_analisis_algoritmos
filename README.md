# Laboratorio 1 — Análisis de Algoritmos

Implementación y análisis experimental de algoritmos recursivos y divide y 
vencerás en Python.

## Contenido
- Subarreglo máximo: fuerza bruta O(n²) vs. divide y vencerás O(n log n)
- Ordenamiento: Merge Sort O(n log n) vs. Insertion Sort O(n²)
- Gráficas comparativas de tiempos de ejecución

## Librerías
- matplotlib
- seaborn

## Cómo ejecutar

1. Clona el repositorio
git clone https://github.com/Alexsanma/laboratorio_1_analisis_algoritmos.git

2. Entra a la carpeta
cd laboratorio_1_analisis_algoritmos

3. Instala las dependencias
pip install -r requirements.txt

4. Abre el cuaderno
jupyter notebook Laboratorio1_AnalisisAlgoritmos.ipynb
